# Bikeshare Project Information

## Description
The goal of the ***Bikeshare Project*** is to use Python to explore data related to bike share systems. It includes one Python file and three data files for **Chicago**, **New York City**, and **Washington**. An **interactive terminal** allows the user to explore this data and receives statistical analyses as a result.
